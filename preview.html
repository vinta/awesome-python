An opinionated list of awesome Python frameworks, libraries, software and resources.

Admin Panels 7

Libraries for administrative interfaces.
  • ajenti - The admin panel your servers deserve.
  • django-grappelli - A jazzy skin for the Django Admin-Interface.
  • flask-admin - Simple and extensible administrative interface framework for Flask.
  • flower - Real-time monitor and web admin for Celery.
  • jet-bridge - Admin panel framework for any application with nice UI.
  • wooey - A Django app which creates automatic web UIs for Python scripts.
  • streamlit - A framework for building dashboards, reports, or chat apps in minutes.

ASGI Servers 3

ASGI-compatible web servers.
  • daphne - A HTTP, HTTP2 and WebSocket protocol server for ASGI.
  • uvicorn - A lightning-fast ASGI server implementation, using uvloop and httptools.
  • hypercorn - An ASGI and WSGI Server based on Hyper libraries.

Asynchronous Programming 8

Libraries for asynchronous, concurrent and parallel execution.
  • asyncio - (Python standard library) Asynchronous I/O, event loop, coroutines and tasks.
  • concurrent.futures - (Python standard library) A high-level interface for asynchronously executing callables.
  • trio - A friendly library for async concurrency and I/O.
  • twisted - An event-driven networking engine.
  • uvloop - Ultra fast asyncio event loop.
  • eventlet - Asynchronous framework with WSGI support.
  • gevent - A coroutine-based Python networking library that uses greenlet.
  • multiprocessing - (Python standard library) Process-based parallelism.

Authentication 2

Libraries for implementing authentication schemes.
  • OAuth
    • authlib - JavaScript Object Signing and Encryption draft implementation.
    • django-allauth - Authentication app for Django that "just works."
    • django-oauth-toolkit - OAuth 2 goodies for Django.
    • oauthlib - A generic and thorough implementation of the OAuth request-signing logic.
  • JWT
    • pyjwt - JSON Web Token implementation in Python.
    • python-jose - A JOSE implementation in Python.

Deep Learning 6

Frameworks for neural networks and deep learning.
  • pytorch - Tensors and Dynamic neural networks in Python with strong GPU acceleration.
  • tensorflow - The most popular Deep Learning framework created by Google.
  • keras - A high-level neural networks library, running on top of TensorFlow.
  • caffe2 - A lightweight, modular, and scalable deep learning framework.
  • mxnet - A deep learning framework designed for efficiency and flexibility.
  • stable-baselines3 - PyTorch implementations of reinforcement learning algorithms.

Web Frameworks 5

Traditional full stack web frameworks.
  • django - The most popular web framework in Python.
  • flask - A microframework for Python.
  • fastapi - A modern, fast web framework for building APIs with Python 3.6+.
  • pyramid - A small, fast, down-to-earth, open source Python web framework.
  • tornado - A Python web framework and asynchronous networking library.
